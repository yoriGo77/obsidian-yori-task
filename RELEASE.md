# Yori Task 发布流程（3 文件版）

本项目当前采用 Obsidian 社区插件常见的三文件发布方式：

- `main.js`
- `manifest.json`
- `styles.css`

## 1. 发版前检查

1. 确认功能改动已完成并本地验证。
2. 确认 `manifest.json` 的 `version` 已更新（例如 `1.0.5`）。
3. 确认 `versions.json` 增加对应版本映射。
4. 若改动涉及文案、交互、样式，建议在桌面端 Obsidian 手动回归：
   - TASK / WEEK / YEAR 三视图
   - 新增、编辑、拖拽、删除、归档
   - 设置页颜色与链接打开策略

## 2. 打包内容

仅提交/发布以下文件到插件目录：

- `main.js`
- `manifest.json`
- `styles.css`

> 当前 `main.js` 已支持“单文件自包含 fallback”模式，即使用户侧缺少 `lib` 目录也可运行。

## 3. 本地安装验证（强烈建议）

将三文件复制到本地 vault：

`<Vault>/.obsidian/plugins/yori-task/`

然后在 Obsidian 中：

1. 关闭并重新启用 `Yori Task` 插件（或重启 Obsidian）。
2. 打开插件面板，检查是否正常加载。
3. 快速验证以下关键行为：
   - WEEK / YEAR 视图区块可滚动且 `+ Add` 可见
   - DONE 列行为正常（勾选进入 DONE、恢复未完成）
   - 归档笔记创建与内部链接打开正常

## 4. 常见问题排查

### 插件无法加载

- 先看 Obsidian 控制台是否有报错（如 `Cannot find module`）。
- 确认插件目录中三文件是同一版本（不要混用旧 `main.js`）。
- 确认 `manifest.json` 的 `id` 为 `yori-task`，且目录名一致。

### 更新后样式异常

- 确认 `styles.css` 已覆盖成功。
- 尝试禁用再启用插件，或重启 Obsidian。

### 版本显示不一致

- 检查 `manifest.json` 与 `versions.json` 是否同时更新。

## 6. 发布前验收清单（可勾选）

> 建议发版前按此清单逐项勾选，避免遗漏。

### A. 基础与版本

- [ ] `manifest.json` 版本号已更新
- [ ] `versions.json` 已新增当前版本映射
- [ ] 三个发布文件已准备好：`main.js`、`manifest.json`、`styles.css`
- [ ] Obsidian 中禁用再启用插件后可正常加载（无报错）

### B. 用户核心流程（TASK / WEEK / YEAR）

- [ ] 可正常新增任务（含 `+ Add` 输入）
- [ ] 可正常编辑任务（回车保存、失焦保存、`Shift+Enter` 换行）
- [ ] 可正常删除任务、复制任务、粘贴任务
- [ ] TASK / WEEK / YEAR 三视图切换正常，无白屏或错位

### C. 拖拽与滚动（高风险）

- [ ] 拖拽任务跨区块移动正常（含滚动场景）
- [ ] 拖拽区块重排正常（2 列 / 3 列 / 单列都验证）
- [ ] 区块拖拽指示线显示正常（行首/行末/中间间隙）
- [ ] 原位拖拽不显示指示线（避免误导）
- [ ] WEEK / YEAR 区块内列表可独立滚动

### D. DONE 与归档

- [ ] 勾选任务进入 DONE 正常
- [ ] DONE 列按“最新完成在最前”显示
- [ ] DONE 超过 30 条后会淘汰最旧项
- [ ] 从 DONE “恢复未完成”可回到正确区块
- [ ] 周归档 / 年归档可创建成功
- [ ] 归档笔记区块标题与当前语言一致

### E. 语言与文案

- [ ] 设置中语言仅有“中文 / English”两项
- [ ] 默认语言为 English
- [ ] 切换语言后界面即时生效（含页签、区块标题、顶部标题）
- [ ] YEAR 顶部仅显示年份（如 `2026`）

### F. 样式一致性（视觉快速检查）

- [ ] 区块标题与任务内容左对齐
- [ ] 任务单行高度、字号符合预期（当前：42 高度 + 16 字号）
- [ ] WEEK 切换箭头无白底、无边框
- [ ] 编辑态输入框背景稳定，不会因鼠标移出“消失”

### G. 发布前最后一步

- [ ] 在一个全新/干净 vault 中完成一次烟测（至少 5 分钟）
- [ ] 确认发布包只含目标版本文件，无旧文件混入
